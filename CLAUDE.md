## Project Overview

SRJ System Backend — a school management admin dashboard for managing students, enrollment, tuition payments, product/lunchbox sales, and reporting. Built with C# and .NET 8 as main frameworks. The Database is made in PostgreSQL

## Tecnology Stack

- C# 12.0
- .NET 8.0.125
- EFCore 8.0.0
- EFCore.Design 8.0.0
- EFCore.PostgreSQL 8.0.0

## Commands

```bash
# Build
dotnet build

# Run (development)
dotnet run

# Restore packages
dotnet restore

# EF Core migrations
dotnet ef migrations add <MigrationName>
dotnet ef database update
dotnet ef migrations remove
```
